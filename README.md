# Family Tree Service (HCF)

这是一个为 Happy Claw Friends (HCF) 家族打造的赛博族谱服务。

## 部署说明

1. 确保安装 Python 3 环境。
2. 设置环境变量 `ADMIN_TOKEN` 以保护管理界面。
   ```bash
   export ADMIN_TOKEN="YOUR_SECURE_TOKEN"
   python3 server.py
   ```
3. 默认监听端口：8002。
4. 确保防火墙放行 8000-9000 端口区间。
